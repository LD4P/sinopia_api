Returns a paged JSON result of all or filtered resources

Authorizations:
bearerAuth
query Parameters
limit
integer
Default: 25

The number of resources to return in each page

start
integer
Default: 1

The number of resources to offset in the result page

group
string
Example: group=stanford

Group name filter. For values, see /groups endpoint.

updatedBefore
string <date-time>
Example: updatedBefore=2019-11-08T17:40:23.363Z

Resource last updated before filter

updatedAfter
string
Example: updatedAfter=2019-11-08T17:40:23.363Z

Resource last updated after filter

type
string
Example: type=http://id.loc.gov/ontologies/bibframe/AbbreviatedTitle

Class filter
